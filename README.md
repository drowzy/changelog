# changelog
Simple tool for generating a markdown changelog based on git commit history.

## Usage
```
Usage: changelog [options]

Options:
  -d, --dir DIR            ./       Directory
  -f, --filename FILENAME  CLOG.md  Filename to output formated logs
  -h, --help
```
